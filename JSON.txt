JSON - javascript object notation
Формат обмена данными между комп в инетернете 
{
    "UserId": 1 ,
    "id": 1 ,
    "title": "Test tittle" ,
    "staus" : {
        "completed": false
    }
}

передача данных в формате json
{"UserId": 1 ,"id": 1 ,"title": "Test tittle" ,"staus" : {"completed": false}}
Передача данных происходит в виде строки

Конвертация JSON в обьект и на оборот
Получив json в виде строки от сервера вы можете передать его как аргумент в
вызове метода 
JSON.parse() и в результате получите джаваскрипт обьект (мы распарсили ее и перевели)
Если наоборот , есть обратная функция JSON.stringify()
